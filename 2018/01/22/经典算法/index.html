冒泡排序

1.比较相邻的两个元素，如果前一个比后一个大，则交换位置。

2.第一轮的时候最后一个元素应该是最大的一个。

3.按照步骤一的方法进行相邻两个元素的比较，这个时候由于最后一个元素已经是最大的了，所以最后一个元素不用比较。 

function sort(arr) {
    for(var i = 0;i<arr.length-1;i++) {
        for(var j = 0;j<arr.length-1-i;j++) {
            if (arr[j]>arr[j+1]) {
                var swap = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = swap;
            }
        }
    }
return arr;
}

快速排序

1.快速排序是对冒泡排序的一种改进，第一趟排序时将数据分成两部分，一部分比另一部分的所有数据都要小。

2.递归调用，在两边都实行快速排序。 

function quickSort(arr) {
    if (arr.length<=1) {
        return arr;
    }
    var index = Math.floor(arr.length/2);
    var indexArr = arr.splice(index,1)[0];
    var left = [];
    var right = [];
    for(var i = 0;i<arr.length;i++) {
        if (arr[i]<=indexArr) {
            left.push(arr[i]);
        }else {
            right.push(arr[i]);
        }
    }
    return quickSort(left).concat([indexArr],quickSort(right));
}

二分查找（折半查找）

1.找到一个中间值，通过与中间值比较，大的放右，小的放在左边。

2.在两边中寻找中间值，持续以上操作，直到找到所在位置为止。 

function bSearch(arr, item, start, end){   
    if(end == undefined){ 
        end = arr.length - 1;  
    }  
    if(start == undefined){  
        start = 0;  
    }  

    var m = Math.floor((start + end) / 2);  

    if(arr[m] == item){
        return m;
    }
    if(start >= end){  
        return false;  
    }   
    if(item < arr[m]){
        return bSearch(arr, item, start, m-1);
    }else{
        return bSearch(arr, item, m+1, end);
    }  
}
